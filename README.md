# BurrAI

### Система для распознавания дефектов речи

### 1. Создайте файл `.env`
### 2. Вставьте свой Telegram API токен в `.env`
```bash
TOKEN="YOUR_TOKEN"
```

### 3. Загрузите все библиотеки с помощью:
```bash
pip install -r requirements.txt
```

### 4. Загрузите uvicorn:
```bash
pip install uvicorn
```

### 5. Запустите FastAPI:
```bash
uvicorn main:app
```
