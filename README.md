# BurrAI - Система для определения речевых дефектов

### Инструкция по установке проекта:
### 1. Создайте файл .env
### 2. Вставьте свой Telegram API токен в .env
TOKEN="YOUR_TOKEN"
### 3. Запустите телеграм бота:
#### Зайдите в папку с телеграм ботом:
cd telegram_bot

#### Для Windows
python bot.py
#### Для Linux & MacOS
python3 bot.py
