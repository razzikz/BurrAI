# BurrAI - Система для определения речевых дефектов

### Инструкция по установке проекта:
### 1. Создайте файл `.env`
### 2. Вставьте свой Telegram API токен в `.env`
```bash
TOKEN="YOUR_TOKEN"
```
### 3. Зайдите в папку с телеграм ботом:
```bash
cd telegram__bot
```
### 4. Запустите телеграм бота:

#### Для Windows
```bash
python bot.py
```
#### Для Linux & MacOS
```bash
python3 bot.py
```
